# PostGIS

Для ознакомления с пространственными индексами используется расширение PostgreSQL – PostGIS,
ввиду его популярности.
Для более глубокого ознакомления с PostGIS рекомендуется [воркшоп](https://postgis.net/workshops/postgis-intro/).

## Docker Compose для PostGIS и pgAdmin

Скопируйте конфигурацию ниже в файл `docker-compose.yml`.

```yaml title="docker-compose.yml"
services:
  postgis:
    container_name: postgis
    image: postgis/postgis
    # platform: linux/amd64
    restart: unless-stopped
    environment:
      POSTGRES_USER: ${POSTGRES_USER:-postgres}
      POSTGRES_PASSWORD: ${POSTGRES_PASSWORD:-postgres}
      POSTGRES_DB: ${POSTGRES_DB:-spatial_lab}
    ports:
      - "${POSTGRES_PORT:-5432}:5432"

  pgadmin:
    container_name: pgadmin
    image: dpage/pgadmin4
    restart: unless-stopped
    environment:
      PGADMIN_DEFAULT_EMAIL: ${PGADMIN_DEFAULT_EMAIL:-admin@example.com}
      PGADMIN_DEFAULT_PASSWORD: ${PGADMIN_DEFAULT_PASSWORD:-admin}
      PGADMIN_CONFIG_SERVER_MODE: "False"
      PGADMIN_CONFIG_MASTER_PASSWORD_REQUIRED: "False"
    ports:
      - "${PGADMIN_PORT:-5050}:80"
```

## Запуск

```bash
docker compose up -d
```

## Доступ

- PostGIS: `localhost:5432`
- pgAdmin: `http://localhost:5050`

## Подключение в pgAdmin

- Host: `postgis` (если pgAdmin запущен в том же compose)
- Port: `5432`
- Username: `postgres`
- Password: `postgres`
- Database: `spatial_lab`
